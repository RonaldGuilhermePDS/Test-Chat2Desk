Follow the instructions to start the project without any problems.

## Installation

1. Clone the Repository
```sh
git clone https://github.com/RonaldGuilhermePDS/Test-Chat2Desk.git
```
2. Create .env file
```sh
cp .env.example .env
```

3. Install Packages
```sh
npm install
```

4. Prepare Prisma
```sh
npm run prepare
```

5. Run Locally
```sh
npm run local
```

or

5. Run with Docker
```sh
npm run up
```
